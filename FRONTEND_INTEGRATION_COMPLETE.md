## 🚀 **How to Run ByteGuardX**

### **Option 1: Full Stack Startup (Recommended)**
```bash
python start_byteguardx_full.py
```

### **Option 2: Manual Startup**
```bash
# Terminal 1 - Backend
python -m byteguardx.api.app

# Terminal 2 - Frontend
npm run dev
```

### **Access URLs:**
- 🌐 **Frontend**: http://localhost:3000
- 🔧 **Backend API**: http://localhost:5000
- 📊 **Dashboard**: http://localhost:3000/dashboard
- 🔌 **Plugin Marketplace**: http://localhost:3000/plugins
- 🔍 **Advanced Scanner**: http://localhost:3000/scan
